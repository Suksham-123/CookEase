 🥘 CookEase

CookEase is an **Android recipe application** that helps users discover delicious recipes based on ingredients they have. It uses the **Spoonacular API** to provide smart recipe suggestions, detailed cooking instructions, and nutritional information — making cooking simple and fun.

## 🛠️ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/Suksham-123/CookEase.git

Open the project in Android Studio

Get a free API key from Spoonacular:
👉 https://spoonacular.com/food-api

Add your API key securely (recommended: local.properties)

SPOONACULAR_API_KEY=your_api_key_here

Build and run the app on an emulator or physical device

📡 API Usage

CookEase uses the Spoonacular API to fetch:

Recipes

Ingredients

Instructions

Nutrition data

⚠️ Make sure to:

Keep your API key private

Follow Spoonacular rate limits
